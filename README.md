# GlowRush_Cosmetics — Interactive Ultra-Realistic Mockup

Это готовый, локально запускаемый макет интернет-магазина косметики GlowRush_Cosmetics.

Что внутри
- Адаптивный, премиум дизайн (пастельные розовые, кремовые, золотые акценты).
- Интерактивный SPA-подобный интерфейс (Главная, Каталог, Страница товара, Корзина, Акции, Контакты).
- Админ-панель для управления товарами, акциями и заказами в реальном времени.
- Сервер на Express + WebSocket (Socket.IO). Данные хранятся в data.json (демонстрация).
- Интерактивные эффекты: hover-анимации, слайдеры, модальные окна, фильтры, анимации при добавлении в корзину.

Требования
- Node.js 18+ и npm

Как запустить локально
1. Сохраните файлы в папке проекта.
2. Установите зависимости:
   npm install
3. Запустите сервер:
   npm start
4. Откройте в браузере:
   - Магазин: http://localhost:3000/
   - Админ-панель: http://localhost:3000/admin.html

Что реализовано
- Главная: крупный баннер, слайдер популярных товаров, блок преимуществ, CTA.
- Каталог: сетка товаров, фильтры по категории и цене, сортировка, карточки с hover-эффектом и кнопкой "Добавить в корзину".
- Страница товара (модал): галерея изображений, описание, статус наличия с цветовой визуализацией, кнопки "Добавить в корзину" и "Купить сейчас".
- Корзина: изменение количества, удаление, оформление заказа (создаёт заказ на сервере).
- Акции: динамические баннеры с переходами.
- Контакты: форма обратной связи, FAQ, карта-заглушка.
- Подвал: ссылки, переключение языков (демо).
- Админ-панель: добавление/редактирование товаров и акций, просмотр заказов. Изменения мгновенно отражаются у клиентов через Socket.IO.

Безопасность и примечания
- Это демонстрационный проект. Админ-эндпойнты не защищены — для продакшена обязательно добавить аутентификацию, валидацию и защиту.
- Для продакшена замените data.json на БД (Postgres/MongoDB), добавьте пагинацию, оптимизацию изображений и интеграцию с платежным шлюзом.
